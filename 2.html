Краткая инфа Тут будет краткая информация по "Html. Будут показаны Тэги для быстрого пользования,чтобы не лезть в интернет.

С чего начинается все:

  • Это команда (html).Затем идет (head) и эти Тэги нужно закрывать и это делается Слэшом (/head)

  • Чтобы переименовать название вкладки мы используем Тэг (title),и он также закрывается (/title)

  • Чтобы также поменять во вкладке иконку мы используем Тэг (link), он как раз таки не закрывается.

Тэги для изменения внешнего вида:

  • (EM)-используется для того чтобы писать курсивом(волнистым шрифтом)

  • (b)-используется для того чтобы писать жирным шрифтом

  • (р)- Используется для того чтобы писать Подзаголовками. Он закрывается

  • (hr)-используется чтобы поставить горизонтальную линию,он не закрывается.

Тэг для вставки ссылки:

  1. a href="вставка ссылки"

  2. (a href="имя файла на который хочем перейти")-это используем для того чтобы открыть другую работу ,например нашу первую работу)

Тэги для списков:

  • (ul)-Используем для нумерования списка,или для элемента (точками)
  • (li)-Используем для начала списка
    1. (ol)используется для номерования списка

Тэг для вставки изображения в текст:

  • (img src="..." alt="")-используем чтобы вставить картинку в текст

Команда для временного отключения Тэга:

  • (ctrl=/)-используем для того чтобы временно отключить команду,но также,со временнем можем ее включить

Тэги для таблицы:

  • (table border="3")-показывает что мы делаем таблицу,он также закрывается
  • (tr)-служит для создание строки,он также закрывается
  • (td)-Предназначен для создания одной ячейки таблицы.

Тэги,для обмена данными:

  • (form)-это база куду он будет писать,поле для написания
  • (label for="название чего либо")-Написать перед полем
  • (value)-чтобы было чтото написано в поле
  • (textarea)-Поле где сам пользователь сможет оставлять сообщение
